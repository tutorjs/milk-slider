# Milk Slider

Milk Slider is a simple slider like a milk.

### Getting Started

```html
<div class="gallery">
  <img src="path/to/your/directory" alt="image">
  <img src="path/to/your/directory" alt="image">
  <img src="path/to/your/directory" alt="image">
</div>

```

```javascript

new MILKSlider({
  target: '.gallery',
  width: 400
})

```
